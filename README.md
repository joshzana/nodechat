A websocket based mobile chat in node.js
===

Components used
---
	- node.js
	- express
	- jade
	- mongodb
	- mongoose ORM
	- socket.IO
	- jquery mobile

To run
---
	- install mongodb
	- create data directory
	- run mongod
	- install node
	- npm install express
	- npm install jade
	- npm install mongoose
	- npm install socket.io
	- node server.js
	- view http://localhost:3000/chats in two or more browsers, and party on the same chat!
